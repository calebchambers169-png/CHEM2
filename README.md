# Chemistry & Physics for Nurse Anesthesia — Exam Review

A browser-based practice quiz for the **Chemistry & Physics for Nurse Anesthesia (2026)** course. **265 questions** across five sections, in multiple-choice and fill-in-the-blank formats, with instant feedback and explanations. Everything runs from a single self-contained HTML file — no install, no build, no internet needed.

## Use it

- **Online:** [calebchambers169-png.github.io/CHEM2](https://calebchambers169-png.github.io/CHEM2/)
- **Offline:** download `index.html` and open it in any browser.

## Features

- **Study by section** — a separate quiz for each of the five topics.
- **Mix sections together** — pick any sections, question types, and length (10 / 25 / 50 / 75 / All).
- **Comprehensive final** — a 100-question mixed exam, 20 per section, randomized, with a per-section score breakdown.
- **Instant feedback** with the correct answer and a short explanation on every question.
- **Forgiving fill-in-the-blank** — ignores capitalization, spacing, and punctuation; accepts common variants.
- **Results & review** — score, by-section breakdown, full answer review, a "missed only" filter, plus retry options.
- **Keyboard shortcuts** — `1`–`4` to answer, `Enter` to advance, arrow keys to navigate.
- **Balanced answer key** — correct answers spread evenly across A/B/C/D.
